How To Run??

Firstly, after starting Apache and MySQL in XAMPP, follow the following steps

1st Step: Extract file
2nd Step: Copy the main project folder
3rd Step: Paste in xampp/htdocs/

Now Connecting Database

4th Step: Open a browser and go to URL “http://localhost/phpmyadmin/”
5th Step: Then, click on the databases tab
6th Step: Create a database naming “ ladyjoy_fs ” and then click on the import tab
7th Step: Click on browse file and select “ ladyjoy_fs .sql” file which is inside “DATABASE” folder
8th Step: Click on go.

After Creating Database,

9th Step: Open a browser and go to URL “http://localhost/cmsphp/”

For Admin Panel

Simply enter Admin’s Login detail

Provide the username and password which is provided inside the “Readme.txt” file.
